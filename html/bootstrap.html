Bootstrap

Bootstrap is een HTML/CSS framework waarmee makkelijk een website met een professioneel uiterlijk gebouwd kan worden.
De Bootstrap framework was origineel ontwikkeld door Twitter, om het front-end development process van twitter makkelijker te maken. Bootstrap heette toen ook 'Twitter Blueprint'.
